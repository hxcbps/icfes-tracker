Modulo 1 - Seccion 1.1

Discreto vs Continuo (decision correcta)

Guia extendida para Saber 11 con criterios, ejemplos y practica aplicada.

Volver al modulo principal

Objetivo didactico

Aprender a decidir si una variable pertenece a $\mathbb{Z}$ (discreta) o $\mathbb{R}$ (continua), y aplicar esa decision para aceptar decimales, redondear o usar techo en problemas de contexto.

Mundo Discreto ($\mathbb{Z}$)

Se cuenta. Hay unidades indivisibles. La respuesta suele ser entera.

  • Personas, buses, sillas, goles, libros.
  • No existe “medio bus” o “media persona”.
  • Si aparece un decimal, se interpreta o se corrige.

Regla tipica

Si se necesita completar grupos o cupos: $\lceil x \rceil$.

Mundo Continuo ($\mathbb{R}$)

Se mide. Hay precision decimal. La respuesta puede ser fraccionaria.

  • Tiempo, distancia, peso, temperatura, volumen.
  • Se expresa con unidades y decimales.
  • Se acepta aproximacion si el problema lo pide.

Regla tipica

Si se mide, conserva decimales y reporta la unidad.

Arbol de decision rapido

  1. Se cuenta o se mide? Contar -> discreto. Medir -> continuo.
  2. Existe unidad minima indivisible? Si, es discreto.
  3. Se puede partir fisicamente? Si, es continuo.
  4. El contexto exige entero? Si, usa techo o redondeo segun el texto.

Ejemplo de contexto

48 estudiantes, bus de 10 -> $48/10=4.8$ buses. Respuesta: $\lceil 4.8 \rceil = 5$.

Casos frontera (Saber 11)

  • Dinero efectivo: discreto por la moneda minima.
  • Dinero digital: continuo por decimales posibles.
  • Promedios: pueden ser reales aunque los datos sean enteros.
  • Capacidad: casi siempre exige entero hacia arriba.

Errores frecuentes

  • Redondear hacia abajo cuando se requiere completar grupos.
  • Tratar tiempo o distancia como enteros por costumbre.
  • Mezclar unidades y terminar con resultados sin sentido.
  • Ignorar la palabra “aprox.” o “como minimo”.

Techo, piso y redondeo

Techo

$\lceil 4.1 \rceil = 5$

Se usa para completar cupos.

Piso

$\lfloor 4.9 \rfloor = 4$

Se usa si se exige no exceder.

Redondeo

$4.4 \to 4$, $4.5 \to 5$

Solo si el enunciado dice “aprox.”

Lenguaje del enunciado (palabras clave)

En Saber 11, las palabras guian la decision. Subraya estas expresiones y aplica la regla correcta.

Techo ($\lceil x \rceil$)

  • "como minimo"
  • "al menos"
  • "no pueden faltar"
  • "cupos completos"

Piso ($\lfloor x \rfloor$)

  • "como maximo"
  • "a lo sumo"
  • "no exceder"
  • "limite permitido"

Redondeo

  • "aprox."
  • "estimado"
  • "redondea a"
  • "cerca de"

Decimales permitidos

  • tiempo, distancia, peso
  • temperatura, volumen
  • velocidad, densidad
  • promedios y tasas

Caso guiado con criterio

"Se deben repartir 250 cuadernos entre 48 estudiantes. Cuantos cuadernos recibe cada uno?"

  1. Se cuenta? Si, cuadernos son discretos.
  2. Operacion: $250/48 = 5.208...$
  3. Interpretacion: no puedes dar 0.208 cuaderno, debes decidir el criterio.
  4. Si es reparto exacto, se entrega 5 y sobran 10 cuadernos.

Conclusion

El contexto define si usas piso, techo o mantienes sobrantes.

Respuesta: $\lceil 101/25 \rceil = 5$.

Mapa mental rapido

Objeto -> Accion -> Tipo

Persona, bus, silla -> contar -> discreto ($\mathbb{Z}$).

Tiempo, distancia, temperatura -> medir -> continuo ($\mathbb{R}$).

Promedio -> resultado de division -> continuo.

Representacion mental (puntos vs linea)

Visualiza la variable en una recta numerica. Si solo hay puntos separados, es discreta. Si hay una linea continua de valores posibles, es continua.

Discreto

o   o   o   o

Valores separados, no hay intermedios.

Continuo

-----------------

Todos los valores entre dos puntos existen.

Mini-caso paso a paso (capacidad)

"Se deben transportar 73 estudiantes. Capacidad del bus: 20."

  1. La variable es numero de buses -> discreta.
  2. Operacion: $73/20 = 3.65$.
  3. Como no pueden faltar, se usa techo.
  4. Resultado: $\lceil 3.65 \rceil = 4$ buses.

Clave

La palabra "completar" implica techo, no redondeo simple.

Mini-caso paso a paso (medicion)

"Un auto recorre 245 km en 3.5 horas. Cual es la velocidad promedio?"

  1. Velocidad es una tasa continua.
  2. Operacion: $245/3.5 = 70$.
  3. Se reporta con unidad: $70\ \text{km/h}$.

Clave

Las tasas casi siempre son continuas.

Unidad minima y cuantizacion

Una variable puede ser discreta porque tiene una unidad minima. Eso se llama cuantizacion: solo existen saltos de tamano fijo.

Ejemplo 1

Moneda minima: $0.01$.

Dinero en efectivo: valores $k(0.01)$.

Ejemplo 2

Buses: unidad minima $1$.

No existe $0.5$ bus.

Ejemplo 3

Tornillos por caja: unidad $1$.

El conteo solo admite enteros.

Idea clave: si puedes escribir los posibles valores como $k\Delta$, la variable es discreta.

Cuando un continuo se vuelve discreto

A veces una variable continua se registra en saltos por regla del problema. En ese caso, se comporta como discreta.

Parqueadero por hora

Tiempo real: $1.2$ h (continuo).

Cobro: $2$ horas (discreto por politica).

Boletas por bloque

Evento vende por bloques de $5$.

Cantidad posible: $0, 5, 10, 15, ...$

Lee el enunciado: si dice "por bloques", "por paquete", o "por hora completa", se discretiza.

Metodo de 3 pasos (rapido y seguro)

  1. Objeto: que cosa estas contando o midiendo?
  2. Unidad: la unidad admite fracciones?
  3. Verbo: el texto pide completar, no exceder o aproximar?

Ejemplo guiado

"Se necesitan estantes para 58 libros. Cada estante soporta 12."

Objeto: estantes (discreto). Unidad: 1 estante. Verbo: completar -> techo.

Operacion: $58/12 = 4.83$, respuesta: $\lceil 4.83 \rceil = 5$.

Banco de situaciones (clasifica rapido)

Discreto o continuo?

  • Numero de vacunas aplicadas
  • Litros de gasolina cargados
  • Minutos de una llamada
  • Personas en una fila
  • Temperatura de un horno

Discreto o continuo?

  • Numero de paginas leidas
  • Altura de una pared
  • Goles en un partido
  • Tiempo de reaccion (s)
  • Promedio de calificaciones

Clave: lo que se mide suele ser continuo, lo que se cuenta suele ser discreto.

Unidades como guia (metodo del rotulo)

Escribe la unidad como un rotulo al lado del numero. Si el rotulo permite fracciones, es continuo. Si no, es discreto.

4.5 kg -> continuo
4.5 buses -> discreto
4.5 horas -> continuo

Relaciones entre conjuntos

Recuerda: los enteros estan dentro de los reales. Es decir, $\mathbb{Z} \subset \mathbb{R}$. Por eso un promedio de enteros puede ser real sin problema.

Ejemplo

$\frac{3 + 4 + 5}{3} = 4$ (entero)

$\frac{3 + 4 + 6}{3} = 4.33$ (real)

Trampas tipicas en Saber 11

  • Te dan decimales y te obligan a pensar si son validos en el contexto.
  • El resultado matematico es correcto, pero el contexto exige ajuste (techo o piso).
  • Confunden promedio con total: el promedio puede ser real aunque los datos sean enteros.
  • La palabra "minimo" siempre cambia la decision.

Autoevaluacion rapida

Responde SI/NO antes de resolver:

  • La variable se cuenta? -> discreto.
  • La variable se mide? -> continuo.
  • Hay palabra clave (minimo/maximo)? -> define techo o piso.
  • El resultado debe ser entero para tener sentido fisico?

Problemas mixtos (dos variables a la vez)

En un mismo problema puedes tener variables discretas y continuas. Identificalas por separado.

Ejemplo

"73 estudiantes viajan durante 4.5 horas en buses de 20."

Variable Tipo Decision
Numero de buses Discreta Techo: $\lceil 73/20 \rceil = 4$
Tiempo total Continua Mantener $4.5$ h

Tip: clasifica cada variable por separado antes de operar con ellas.

Guia rapida de decision (4 escenarios)

Completar grupos

"Necesarios", "cupos completos", "no pueden faltar".

Usa techo: $\lceil x \rceil$.

No exceder limite

"A lo sumo", "maximo", "sin pasarse".

Usa piso: $\lfloor x \rfloor$.

Mediciones reales

Tiempo, distancia, peso, volumen.

Conserva decimales y unidad.

Aproximacion pedida

"Aprox.", "redondea a", "cerca de".

Redondea segun la consigna.

Decimales con contexto (dinero, tiempo y conteo)

Un decimal puede ser valido o invalido segun el objeto. La misma cifra significa cosas distintas si se trata de medir o de contar.

Dinero

$\$2{,}500.75$ es valido en dinero digital.

En efectivo depende de la moneda minima.

Tiempo

$2.5$ horas es valido (2 h 30 min).

Es continuo, se puede fraccionar.

Conteo

$2.5$ personas no es valido.

Se ajusta con techo o piso.

Regla corta: medir -> decimal permitido, contar -> ajustar.

Metodo de verificacion (sentido fisico)

Antes de elegir una opcion, valida si el resultado tiene sentido real.

  1. Escribe la unidad al lado del numero (ej: $4.2$ buses).
  2. Pregunta: "Se puede partir ese objeto?"
  3. Si la respuesta es no, ajusta con techo o piso.
  4. Revisa si la palabra clave lo confirma (minimo/maximo/aprox.).

Chequeo rapido

Si al leer la respuesta suena raro ("2.3 estudiantes"), es discreto.

Errores finos (los que mas confunden)

  • Usar redondeo cuando el texto exige completar (debes usar techo).
  • Ignorar el verbo del enunciado: "al menos" cambia la decision.
  • Tratar promedios como enteros por costumbre.
  • Olvidar la unidad al final, lo que oculta el tipo de variable.

Glosario rapido

Discreto

Solo valores separados. Se cuenta.

Continuo

Todos los valores intermedios. Se mide.

Techo

Menor entero mayor o igual: $\lceil x \rceil$.

Piso

Mayor entero menor o igual: $\lfloor x \rfloor$.

Lectura de graficas simples

Clasifica la variable del eje $x$ como discreta o continua segun la grafica descrita.

Grafica 1

Solo puntos separados en $x = 1,2,3,4$.

Respuesta: discreta (conteo).

Grafica 2

Linea continua entre $x=0$ y $x=5$.

Respuesta: continua (medicion).

Miniguia de conversiones (para no fallar)

Las conversiones no cambian el tipo de variable, pero si el numero. Conserva la unidad.

Tiempo

$0.5$ h $= 30$ min.

Sigue siendo continuo.

Distancia

$1.2$ km $= 1200$ m.

Sigue siendo continuo.

Conteo

12 docenas $= 144$ unidades.

Sigue siendo discreto.

Reto Saber 11 (contexto real)

Un colegio debe imprimir 520 guias. Cada paquete trae 48. Ademas, cada estudiante debe caminar 1.8 km en una salida pedagogica. Responde:

  1. Cuantos paquetes de guias se necesitan?
  2. La distancia de 1.8 km es discreta o continua?

Solucion guiada

Paquetes: $520/48 = 10.83$ -> $\lceil 10.83 \rceil = 11$.

Distancia: continua (se mide y admite decimales).

Mismo numero, distinta unidad

El valor decimal no decide el tipo de variable. La unidad es la que manda.

Numero Unidad Tipo Justificacion
3.5 horas Continua Se mide y admite fracciones.
3.5 buses Discreta No se puede partir un bus.
3.5 kg Continua Peso se mide con precision.
3.5 estudiantes Discreta Conteo indivisible.

Tiempo en decimales (conversion rapida)

Cuando aparece un decimal en horas, puedes pasarlo a minutos multiplicando por 60.

Ejemplo 1

$2.5$ h $= 2$ h $30$ min.

Ejemplo 2

$1.25$ h $= 1$ h $15$ min.

Ejemplo 3

$0.6$ h $= 36$ min.

Esto no cambia el tipo de variable: sigue siendo continua.

Comparativo techo vs piso (casos rapidos)

Enunciado Operacion Decision
"Se requieren buses para 89 personas. Cada bus lleva 20." $89/20 = 4.45$ Techo: $\lceil 4.45 \rceil = 5$
"Como maximo caben 70 personas en grupos de 8." $70/8 = 8.75$ Piso: $\lfloor 8.75 \rfloor = 8$
"Se necesitan paquetes completos de 12 para 95 items." $95/12 = 7.91$ Techo: $\lceil 7.91 \rceil = 8$
"Solo se pueden vender 40 combos maximo. Cada combo usa 6 unidades." $40/6 = 6.66$ Piso: $\lfloor 6.66 \rfloor = 6$

Como elegir en opciones multiple (estrategia)

Calculo: $102/24 = 4.25$.

Opcion con 4.25 es distractor. La correcta es $\lceil 4.25 \rceil = 5$.

Promedios: el error mas comun

El promedio es una division. Aunque los datos sean enteros, el resultado puede ser real.

Ejemplo 1

Datos: 3, 4, 5.

Promedio: $(3+4+5)/3 = 4$ (entero).

Ejemplo 2

Datos: 3, 4, 6.

Promedio: $(3+4+6)/3 = 4.33$ (real).

Nunca redondees un promedio si el enunciado no lo pide.

Reto con presupuesto (piso vs techo)

El presupuesto introduce un limite. Eso suele implicar piso.

Problema

Hay $500{,}000$ para comprar paquetes de $78{,}000$. Cuantos paquetes maximo?

Operacion: $500000/78000 = 6.41$ -> piso: $\lfloor 6.41 \rfloor = 6$.

Clave: "maximo" y "no exceder" implican piso.
