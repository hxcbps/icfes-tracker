Modulo 1 - Seccion 1.1

Discreto vs Continuo (decision correcta)

Guia extendida para Saber 11 con criterios, ejemplos y practica aplicada.

Volver al modulo principal

Objetivo didactico

Aprender a decidir si una variable pertenece a $\mathbb{Z}$ (discreta) o $\mathbb{R}$ (continua), y aplicar esa decision para aceptar decimales, redondear o usar techo en problemas de contexto.

Mundo Discreto ($\mathbb{Z}$)

Se cuenta. Hay unidades indivisibles. La respuesta suele ser entera.

  • Personas, buses, sillas, goles, libros.
  • No existe “medio bus” o “media persona”.
  • Si aparece un decimal, se interpreta o se corrige.

Regla tipica

Si se necesita completar grupos o cupos: $\lceil x \rceil$.

Mundo Continuo ($\mathbb{R}$)

Se mide. Hay precision decimal. La respuesta puede ser fraccionaria.

  • Tiempo, distancia, peso, temperatura, volumen.
  • Se expresa con unidades y decimales.
  • Se acepta aproximacion si el problema lo pide.

Regla tipica

Si se mide, conserva decimales y reporta la unidad.

Arbol de decision rapido

  1. Se cuenta o se mide? Contar -> discreto. Medir -> continuo.
  2. Existe unidad minima indivisible? Si, es discreto.
  3. Se puede partir fisicamente? Si, es continuo.
  4. El contexto exige entero? Si, usa techo o redondeo segun el texto.

Ejemplo de contexto

48 estudiantes, bus de 10 -> $48/10=4.8$ buses. Respuesta: $\lceil 4.8 \rceil = 5$.

Casos frontera (Saber 11)

  • Dinero efectivo: discreto por la moneda minima.
  • Dinero digital: continuo por decimales posibles.
  • Promedios: pueden ser reales aunque los datos sean enteros.
  • Capacidad: casi siempre exige entero hacia arriba.

Errores frecuentes

  • Redondear hacia abajo cuando se requiere completar grupos.
  • Tratar tiempo o distancia como enteros por costumbre.
  • Mezclar unidades y terminar con resultados sin sentido.
  • Ignorar la palabra “aprox.” o “como minimo”.

Techo, piso y redondeo

Techo

$\lceil 4.1 \rceil = 5$

Se usa para completar cupos.

Piso

$\lfloor 4.9 \rfloor = 4$

Se usa si se exige no exceder.

Redondeo

$4.4 \to 4$, $4.5 \to 5$

Solo si el enunciado dice “aprox.”

Lenguaje del enunciado (palabras clave)

En Saber 11, las palabras guian la decision. Subraya estas expresiones y aplica la regla correcta.

Techo ($\lceil x \rceil$)

  • "como minimo"
  • "al menos"
  • "no pueden faltar"
  • "cupos completos"

Piso ($\lfloor x \rfloor$)

  • "como maximo"
  • "a lo sumo"
  • "no exceder"
  • "limite permitido"

Redondeo

  • "aprox."
  • "estimado"
  • "redondea a"
  • "cerca de"

Decimales permitidos

  • tiempo, distancia, peso
  • temperatura, volumen
  • velocidad, densidad
  • promedios y tasas

Caso guiado con criterio

"Se deben repartir 250 cuadernos entre 48 estudiantes. Cuantos cuadernos recibe cada uno?"

  1. Se cuenta? Si, cuadernos son discretos.
  2. Operacion: $250/48 = 5.208...$
  3. Interpretacion: no puedes dar 0.208 cuaderno, debes decidir el criterio.
  4. Si es reparto exacto, se entrega 5 y sobran 10 cuadernos.

Conclusion

El contexto define si usas piso, techo o mantienes sobrantes.

Mini quiz tipo Saber 11

Pregunta A

Un bus tiene capacidad de 25 personas. Hay 101 personas. Cuantos buses se requieren?

Respuesta: $\lceil 101/25 \rceil = 5$.

Pregunta B

Un atleta corre 2.75 km. Es valido reportar 2.75?

Si, distancia es continua.

Pregunta C

El promedio de 3, 4 y 5 es 4.0. El promedio de 3, 4 y 6 es 4.33. Es valido?

Si, el promedio puede ser real.

Mapa mental rapido

Objeto -> Accion -> Tipo

Persona, bus, silla -> contar -> discreto ($\mathbb{Z}$).

Tiempo, distancia, temperatura -> medir -> continuo ($\mathbb{R}$).

Promedio -> resultado de division -> continuo.

Representacion mental (puntos vs linea)

Visualiza la variable en una recta numerica. Si solo hay puntos separados, es discreta. Si hay una linea continua de valores posibles, es continua.

Discreto

o   o   o   o

Valores separados, no hay intermedios.

Continuo

-----------------

Todos los valores entre dos puntos existen.

Mini-caso paso a paso (capacidad)

"Se deben transportar 73 estudiantes. Capacidad del bus: 20."

  1. La variable es numero de buses -> discreta.
  2. Operacion: $73/20 = 3.65$.
  3. Como no pueden faltar, se usa techo.
  4. Resultado: $\lceil 3.65 ceil = 4$ buses.

Clave

La palabra "completar" implica techo, no redondeo simple.

Mini-caso paso a paso (medicion)

"Un auto recorre 245 km en 3.5 horas. Cual es la velocidad promedio?"

  1. Velocidad es una tasa continua.
  2. Operacion: $245/3.5 = 70$.
  3. Se reporta con unidad: $70\ ext{km/h}$.

Clave

Las tasas casi siempre son continuas.

Lista de chequeo de redondeo

Usa techo si...

  • Hay cupos o grupos completos.
  • Dice "al menos" o "como minimo".
  • El decimal implicaria faltar.

Usa piso si...

  • Dice "como maximo".
  • Hay limite de presupuesto.
  • No se puede exceder un tope.

Si no hay palabra clave, conserva el decimal o pregunta por el contexto.

Practica guiada

1) Se necesitan 3.2 camiones para transportar carga. Que respuesta es valida?

Discreto -> se necesita 4 camiones (techo).

2) El recorrido es de 12.5 km. Es valido reportar decimales?

Si, distancia es continua.

3) El promedio de notas enteras da 3.7. Es invalido?

No, el promedio puede ser real aunque los datos sean enteros.

4) Se necesitan 40 sillas y hay 38. Cuantas faltan?

Discreto -> faltan 2 sillas.

5) Una botella tiene 1.25 L. Es valido reportar decimales?

Si, volumen es continuo.

Practica adicional (sin respuesta)

  1. Hay 97 estudiantes y buses de 30. Cuantos buses se requieren?
  2. Un tramo mide 2.75 km. Es discreto o continuo?
  3. Se reparte una pizza en 8 partes iguales. Que tipo de variable es?
  4. Se compran 3.6 kg de arroz. Es discreto o continuo?
  5. Un promedio de 4 datos enteros da 2.75. Es valido?

Sugerencia: escribe la unidad al lado de cada numero antes de decidir.

Banco de situaciones (clasifica rapido)

Discreto o continuo?

  • Numero de vacunas aplicadas
  • Litros de gasolina cargados
  • Minutos de una llamada
  • Personas en una fila
  • Temperatura de un horno

Discreto o continuo?

  • Numero de paginas leidas
  • Altura de una pared
  • Goles en un partido
  • Tiempo de reaccion (s)
  • Promedio de calificaciones

Clave: lo que se mide suele ser continuo, lo que se cuenta suele ser discreto.

Unidades como guia (metodo del rotulo)

Escribe la unidad como un rotulo al lado del numero. Si el rotulo permite fracciones, es continuo. Si no, es discreto.

4.5 kg -> continuo
4.5 buses -> discreto
4.5 horas -> continuo

Relaciones entre conjuntos

Recuerda: los enteros estan dentro de los reales. Es decir, $\\mathbb{Z} \\subset \\mathbb{R}$. Por eso un promedio de enteros puede ser real sin problema.

Ejemplo

$\\frac{3 + 4 + 5}{3} = 4$ (entero)

$\\frac{3 + 4 + 6}{3} = 4.33$ (real)

Trampas tipicas en Saber 11

  • Te dan decimales y te obligan a pensar si son validos en el contexto.
  • El resultado matematico es correcto, pero el contexto exige ajuste (techo o piso).
  • Confunden promedio con total: el promedio puede ser real aunque los datos sean enteros.
  • La palabra "minimo" siempre cambia la decision.

Autoevaluacion rapida

Responde SI/NO antes de resolver:

  • La variable se cuenta? -> discreto.
  • La variable se mide? -> continuo.
  • Hay palabra clave (minimo/maximo)? -> define techo o piso.
  • El resultado debe ser entero para tener sentido fisico?

Checklist del tutor

  • Subraya el objeto: se cuenta o se mide?
  • Escribe la unidad y revisa si puede fraccionarse.
  • Si hay cupos o personas, usa techo.
  • Si hay mediciones, conserva decimales y la unidad.
