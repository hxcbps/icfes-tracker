Modulo 1 - Seccion 1.1

Discreto vs Continuo (decision correcta)

Guia extendida para Saber 11 con criterios, ejemplos y practica aplicada.

Volver al modulo principal

Objetivo didactico

Aprender a decidir si una variable pertenece a $\mathbb{Z}$ (discreta) o $\mathbb{R}$ (continua), y aplicar esa decision para aceptar decimales, redondear o usar techo en problemas de contexto.

Mundo Discreto ($\mathbb{Z}$)

Se cuenta. Hay unidades indivisibles. La respuesta suele ser entera.

  • Personas, buses, sillas, goles, libros.
  • No existe “medio bus” o “media persona”.
  • Si aparece un decimal, se interpreta o se corrige.

Regla tipica

Si se necesita completar grupos o cupos: $\lceil x \rceil$.

Mundo Continuo ($\mathbb{R}$)

Se mide. Hay precision decimal. La respuesta puede ser fraccionaria.

  • Tiempo, distancia, peso, temperatura, volumen.
  • Se expresa con unidades y decimales.
  • Se acepta aproximacion si el problema lo pide.

Regla tipica

Si se mide, conserva decimales y reporta la unidad.

Arbol de decision rapido

  1. Se cuenta o se mide? Contar -> discreto. Medir -> continuo.
  2. Existe unidad minima indivisible? Si, es discreto.
  3. Se puede partir fisicamente? Si, es continuo.
  4. El contexto exige entero? Si, usa techo o redondeo segun el texto.

Ejemplo de contexto

48 estudiantes, bus de 10 -> $48/10=4.8$ buses. Respuesta: $\lceil 4.8 \rceil = 5$.

Casos frontera (Saber 11)

  • Dinero efectivo: discreto por la moneda minima.
  • Dinero digital: continuo por decimales posibles.
  • Promedios: pueden ser reales aunque los datos sean enteros.
  • Capacidad: casi siempre exige entero hacia arriba.

Errores frecuentes

  • Redondear hacia abajo cuando se requiere completar grupos.
  • Tratar tiempo o distancia como enteros por costumbre.
  • Mezclar unidades y terminar con resultados sin sentido.
  • Ignorar la palabra “aprox.” o “como minimo”.

Techo, piso y redondeo

Techo

$\lceil 4.1 \rceil = 5$

Se usa para completar cupos.

Piso

$\lfloor 4.9 \rfloor = 4$

Se usa si se exige no exceder.

Redondeo

$4.4 \to 4$, $4.5 \to 5$

Solo si el enunciado dice “aprox.”

Practica guiada

1) Se necesitan 3.2 camiones para transportar carga. Que respuesta es valida?

Discreto -> se necesita 4 camiones (techo).

2) El recorrido es de 12.5 km. Es valido reportar decimales?

Si, distancia es continua.

3) El promedio de notas enteras da 3.7. Es invalido?

No, el promedio puede ser real aunque los datos sean enteros.

Checklist del tutor

  • Subraya el objeto: se cuenta o se mide?
  • Escribe la unidad y revisa si puede fraccionarse.
  • Si hay cupos o personas, usa techo.
  • Si hay mediciones, conserva decimales y la unidad.
